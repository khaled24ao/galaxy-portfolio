Galaxy OS is a next-generation, highly interactive 3D portfolio built for the future. It drops the traditional scrolling website approach in favor of a **AAA cinematic space experience**. Navigate a procedural solar system, dive into gravitational anomalies, and interact with a highly polished Cyberpunk/Sci-Fi HUD.

## 🚀 Features

- **Cinematic Experience:** 25-second breathtaking introductory sequence simulating the Big Bang and a warp jump into a solar system.
- **Interactive 3D Galaxy:** Fully explorable solar system. Click on planets to target them and view individual projects.
- **Interstellar Black Hole:** A complex shader-based black hole featuring an accretion disk and gravitational lensing.
- **AAA Cyberpunk UI:** Custom UI overlay with procedural decryption effects, clip-path geometries, holographic glow, and interactive HUD tracking.
- **Immersive Audio:** Fully integrated sound design featuring symphonic space scores, warp jump sound effects, and UI feedback clicks using `Howler.js`.

## 🛠️ Technology Stack

- **Framework:** React 19 + Vite
- **3D Engine:** Three.js + React Three Fiber (`@react-three/fiber`, `@react-three/drei`)
- **Animations:** Framer Motion + GSAP
- **Post-Processing:** Bloom, Chromatic Aberration, and Vignette effects (`@react-three/postprocessing`)
- **Audio:** Howler.js

## 💻 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/khaled24ao/galaxy-os.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the local dev server:
   ```bash
   npm run dev
   ```
