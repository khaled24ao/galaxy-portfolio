# Sound Files for NEXUS 2050

This directory should contain the following audio files (MP3 or WAV):

## Required Sounds
- `ambient-space.mp3` - 3 min loop, dark ambient space background
- `hover-planet.mp3` - 0.3s subtle beep
- `click-planet.mp3` - 0.2s confirmation click
- `hud-open.mp3` - 0.5s whoosh + UI sounds
- `hud-close.mp3` - 0.3s swoosh
- `glitch-transition.mp3` - 0.8s digital glitch effect
- `meteor-explosion.mp3` - 0.4s explosion

## Optional
- `boot-sequence.mp3` - 2s computer boot sound

## Placeholder
If you don't have sound files yet, place minimal silent MP3s here. The AudioManager gracefully handles missing files.

## Audio Sources
You can create/free download from:
- Freesound.org
- Pixabay Audio
- ZapSplat
- Or generate with tools like Mubert, AIVA, etc.
